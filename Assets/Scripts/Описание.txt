Скример.
Screamer Active - Включает объект на сцене при входе в триггер.
Screamer Destroy - Уничтожает объект на сцене и убирает его их иерархии.
Screamer False - Отключает объект на сцене при входе в триггер (остается в иерархии).

Как применять скрипт?
- Вешаем скрипт на триггер, попутно его создав GameObject - 3D object - Cube. В настройке куба убираем mesh render и ставим box colliders галочку is trigger.


ИИ.
Ai_Ray - Включает передвижение монстра по карте с обоходом препятствий.
Как применить скрипт?
- Скрипт вешаем на монстра и создаем на нём Nav Mesh Agent, нажав на Add component в инспекторе (выбрав монстра). Далее, создаем навигационную сетку, (не забываем 
ставить галочку statik для всех неподвижных объектов, в том числе и на terrain)  для этого нажимаем на windows - AI - Navigations. 
Нажимаем Bake, затем ещё раз Bake. Как только будет создана навигационная сетка, карта покроется голубыми зонами.
На плеере ставим тег Player, чтобы монстр знал за кем охотиться.
